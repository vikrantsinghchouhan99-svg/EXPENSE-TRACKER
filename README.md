# EXPENSE-TRACKER
we build a fully functional expense tracking application that allows users to add, delete, and manage expenses, calculate total balance in real time, and store data locally using LocalStorage.
🛠️ What You’ll Learn

✔ Structuring a project using HTML & CSS
✔ Building an interactive UI with JavaScript
✔ DOM manipulation and event handling
✔ Adding & deleting expense items dynamically
✔ Calculating income, expenses & balance
✔ Using LocalStorage for data persistence
✔ Writing clean, reusable, beginner-friendly code
